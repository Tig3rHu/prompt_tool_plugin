# 智能提示词工具插件 - 测试指南

## 🧪 测试步骤

### 1. 安装插件
按照 `INSTALL.md` 中的步骤安装插件到Chrome浏览器。

### 2. 访问测试页面
打开以下任一网站进行测试：
- [ChatGPT](https://chat.openai.com)
- [DeepSeek](https://chat.deepseek.com)

### 3. 基本功能测试

#### 3.1 悬浮窗显示测试
- ✅ 页面加载后应自动显示悬浮窗
- ✅ 悬浮窗应显示在页面右上角
- ✅ 悬浮窗应包含提示词列表

#### 3.2 提示词插入测试
1. 点击任意提示词的"插入"按钮
2. 检查是否成功插入到输入框
3. 如果插入失败，会显示调试信息

#### 3.3 搜索功能测试
1. 在搜索框中输入关键词
2. 检查提示词列表是否实时过滤
3. 清空搜索框，检查是否恢复完整列表

#### 3.4 拖拽功能测试
1. 点击悬浮窗标题栏并拖拽
2. 检查是否可以移动到任意位置
3. 检查是否限制在视窗范围内

#### 3.5 最小化/展开测试
1. 点击标题栏的"−"按钮
2. 检查悬浮窗是否最小化
3. 点击"+"按钮检查是否展开

### 4. 故障排除测试

#### 4.1 输入框检测失败
如果插入功能不工作：

1. **查看调试信息**
   - 插入失败时会显示调试面板
   - 查看检测到的输入元素列表
   - 检查控制台日志

2. **手动选择输入框**
   - 点击调试面板中的"手动选择输入框"按钮
   - 点击页面上的输入框
   - 再次尝试插入提示词

3. **检查控制台日志**
   - 按F12打开开发者工具
   - 查看Console标签页的日志
   - 寻找"正在查找输入框"相关日志

#### 4.2 常见问题解决

**问题：悬浮窗不显示**
- 解决方案：刷新页面，或点击浏览器工具栏中的插件图标

**问题：提示词插入失败**
- 解决方案：使用手动选择功能，或检查输入框是否已加载

**问题：搜索功能不工作**
- 解决方案：确保输入了有效的搜索关键词

### 5. 高级功能测试

#### 5.1 提示词管理
1. 点击插件图标打开管理界面
2. 测试添加、编辑、删除提示词
3. 测试导入导出功能

#### 5.2 多平台兼容性
- 测试ChatGPT不同版本的兼容性
- 测试DeepSeek的兼容性
- 测试其他可能的AI对话平台

## 🔍 调试技巧

### 1. 使用浏览器控制台
```javascript
// 查看插件实例
console.log(window.promptTool);

// 手动触发插入
window.promptTool.insertPrompt(1);

// 查看当前检测到的输入框
window.promptTool.findInputElement();
```

### 2. 检查页面元素
```javascript
// 查找所有可能的输入框
document.querySelectorAll('textarea, input[type="text"], div[contenteditable="true"]');

// 检查特定选择器
document.querySelector('textarea[placeholder*="Message"]');
```

### 3. 手动测试插入功能
```javascript
// 获取输入框
const input = document.querySelector('textarea[placeholder*="Message"]');

// 手动插入文本
input.value = "测试文本";
input.dispatchEvent(new Event('input', { bubbles: true }));
```
