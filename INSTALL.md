# 智能提示词工具插件 - 安装指南

## 🚀 快速安装

### 步骤1：下载插件
确保你已经下载了完整的插件文件夹 `prompt_tool_plugin`

### 步骤2：打开Chrome扩展程序页面
1. 打开Chrome浏览器
2. 在地址栏输入：`chrome://extensions/`
3. 按回车键进入扩展程序管理页面

### 步骤3：启用开发者模式
1. 在扩展程序页面右上角找到"开发者模式"开关
2. 点击开关，启用开发者模式

### 步骤4：加载插件
1. 点击"加载已解压的扩展程序"按钮
2. 在弹出的文件选择器中，选择 `prompt_tool_plugin` 文件夹
3. 点击"选择文件夹"

### 步骤5：验证安装
1. 插件应该出现在扩展程序列表中
2. 确保插件状态为"已启用"
3. 浏览器工具栏应该出现插件图标

## 🎯 使用方法

### 基本使用流程

1. **访问支持的网站**
   - 打开 [ChatGPT](https://chat.openai.com) 或 [DeepSeek](https://chat.deepseek.com)

2. **激活插件**
   - 插件会自动检测页面并显示悬浮窗
   - 如果未显示，点击浏览器工具栏中的插件图标

3. **使用提示词**
   - 在悬浮窗中浏览提示词列表
   - 使用搜索框快速查找
   - 点击"插入"按钮将提示词插入到输入框

### 功能详解

#### 🔍 搜索功能
- 在搜索框中输入关键词
- 支持搜索标题、内容、分类和标签
- 实时显示搜索结果

#### 📝 管理提示词
- 点击插件图标打开管理界面
- 可以添加、编辑、删除提示词
- 支持按分类筛选
- 可以导入导出提示词库

#### 🖱️ 拖拽移动
- 点击悬浮窗标题栏并拖拽
- 可以移动到屏幕任意位置
- 自动限制在视窗范围内

#### 📱 最小化/展开
- 点击标题栏的"−"按钮最小化
- 点击"+"按钮展开
- 点击"×"按钮关闭悬浮窗

## 🛠️ 故障排除

### 常见问题

**Q: 插件图标没有出现在工具栏？**
A: 
1. 检查扩展程序页面中插件是否已启用
2. 尝试刷新页面
3. 重新加载插件

**Q: 悬浮窗没有显示？**
A: 
1. 确保访问的是支持的网站（ChatGPT或DeepSeek）
2. 刷新页面重试
3. 点击插件图标手动激活

**Q: 提示词插入失败？**
A: 
1. 确保输入框已加载完成
2. 手动点击输入框后再尝试插入
3. 检查页面是否完全加载

**Q: 搜索功能不工作？**
A: 
1. 确保输入了有效的搜索关键词
2. 尝试清空搜索框重新搜索
3. 检查提示词数据是否正常加载

### 重新安装步骤

如果遇到问题，可以尝试重新安装：

1. 在扩展程序页面找到插件
2. 点击"移除"按钮删除插件
3. 重新按照安装步骤加载插件

## 📞 获取帮助

如果遇到其他问题：

1. 查看浏览器控制台是否有错误信息
2. 检查插件权限设置
3. 尝试在隐私模式下测试
4. 联系开发者获取支持

## 🔄 更新插件

当有新版本时：

1. 下载新版本文件
2. 在扩展程序页面点击"重新加载"按钮
3. 或者删除旧版本后重新安装
