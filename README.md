# 智能提示词工具插件

一个功能强大的浏览器扩展，专为ChatGPT和DeepSeek等AI对话平台设计，提供便捷的提示词管理功能。

## ✨ 主要功能

### 🔍 自动页面检测
- 自动识别ChatGPT和DeepSeek页面
- 智能适配不同平台的输入框
- 无需手动配置，即插即用

### 🪟 美观悬浮窗
- 现代化设计，美观易用
- 渐变背景和流畅动画
- 响应式布局，适配不同屏幕

### 📝 提示词管理
- 内置丰富的默认提示词库
- 支持自定义添加、编辑、删除提示词
- 智能分类和标签系统
- 支持导入导出功能

### 🔍 智能搜索
- 实时搜索提示词标题、内容、分类和标签
- 支持模糊匹配和关键词高亮
- 快速定位所需提示词

### 🖱️ 拖拽移动
- 自由拖拽悬浮窗到任意位置
- 自动限制在视窗范围内
- 记住位置，下次打开保持原位置

### 📱 最小化/展开
- 一键最小化节省屏幕空间
- 快速展开恢复完整功能
- 状态记忆，操作便捷

### ⚡ 一键插入
- 点击即可自动插入到输入框
- 智能识别不同平台的输入框
- 支持复制到剪贴板功能

## 🚀 安装方法

### 方法一：开发者模式安装（推荐）

1. 下载或克隆此项目到本地
2. 打开Chrome浏览器，进入扩展程序页面（`chrome://extensions/`）
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹
6. 扩展安装完成！

### 方法二：打包安装

1. 在扩展程序页面点击"打包扩展程序"
2. 选择项目文件夹
3. 生成.crx文件后拖拽到扩展程序页面安装

## 📖 使用指南

### 基本使用

1. **访问支持的网站**
   - 打开ChatGPT（chat.openai.com 或 chatgpt.com）
   - 或打开DeepSeek（chat.deepseek.com）

2. **激活插件**
   - 插件会自动检测页面并显示悬浮窗
   - 如果未显示，点击浏览器工具栏中的插件图标

3. **使用提示词**
   - 在悬浮窗中浏览提示词列表
   - 使用搜索框快速查找
   - 点击"插入"按钮将提示词插入到输入框
   - 点击"复制"按钮复制到剪贴板

### 高级功能

1. **管理提示词**
   - 点击插件图标打开管理界面
   - 添加、编辑、删除自定义提示词
   - 按分类筛选和搜索
   - 导入导出提示词库

2. **自定义设置**
   - 调整悬浮窗位置
   - 设置默认分类
   - 配置快捷键

## 🔧 开发说明

### 项目结构

```
prompt_tool_plugin/
├── manifest.json          # 扩展配置文件
├── content.js             # 内容脚本
├── styles.css             # 样式文件
├── popup.html             # 弹窗页面
├── popup.js               # 弹窗脚本
├── prompt-manager.html    # 提示词管理页面
├── prompt-manager.js      # 管理页面脚本
├── icons/                 # 图标文件夹
│   ├── icon16.png
│   ├── icon48.png
│   └── icon128.png
└── README.md              # 说明文档
```

### 核心类说明

- **PromptTool**: 主插件类，负责页面检测、悬浮窗创建和事件处理
- **PopupManager**: 弹窗管理类，处理扩展图标点击事件
- **PromptManager**: 提示词管理类，提供CRUD操作和导入导出功能

### 开发环境

- Chrome Extension Manifest V3
- 现代浏览器支持（Chrome 88+）
- 无需额外构建工具

## 🆘 常见问题

### Q: 插件在某些页面不工作？
A: 请确保页面URL包含支持的域名，并刷新页面重试。

### Q: 提示词插入失败？
A: 请确保输入框已加载完成，可以尝试手动点击输入框后再插入。

### Q: 如何备份我的提示词？
A: 在管理页面点击"导出"按钮，可以下载JSON格式的备份文件。

### Q: 支持其他AI平台吗？
A: 目前主要支持ChatGPT和DeepSeek，其他平台可以通过修改manifest.json中的host_permissions来添加支持。
